### 🚨 优先级最高：昨日系统崩溃热修复 (Hotfix)

在进入常规复习前，必须先打补丁，这是工程师的铁律。

- **热修复 1（切面污染修复）**：盲打一遍今天写错的 `@time_logger` 装饰器。
- **验收标准**：必须带参数，必须正确给 `@wraps` 传入 `func`，如果捕获异常，必须使用单纯的 `raise` 原样抛出，绝不能再写 `raise "字符串"`。

```py
from functools import wraps
import time


def time_logger(t=None):
    def decorate(func):
        @wraps(func)
        def wrapper(*arg, **kwargs):
            try:
                start_time = time.time()
                result = func(*arg, **kwargs)
                print(f"耗时:{time.time()-start_time}")
            except Exception as e:
                print(f"发生了错误:{e}")
                raise

        return wrapper

    return decorate

```

- **热修复 2（资源泄漏修复）**：默写今天那个会耗尽文件描述符的 TCP 服务端 `while True` 循环。
- **验收标准**：在处理完客户端请求后，必须加上 `socket_ap.close()`。

```py

import socket

server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

server.bind(("127.0.0.1", 8888))

server.listen(5)

while True:
    socket_ap, socket_info = server.accept()
    result = socket_ap.recv(1024).decode("utf-8")
    if result == "exit":
        socket_ap.close()
        server.close()
        break

```
